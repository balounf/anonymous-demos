This page demonstrates how to start an AxonIvy process from a web-page managed in an external system as an anonymous user. The page is served by the Apache web server and contains an HTML form. When the form is submitted, the data is sent to AxonIvy via a POST request.
Note: The user is only identified by e-mail address. No AxonIvy user will be created. This page is configured to call a process in the AxonIvy Designer.
